# Frontend — Configurador de Painéis

SPA em **React 19**, **TypeScript** e **Vite**. Código em `src/`; módulos de negócio em `src/modules/`.

> **Portfólio:** UI do **wizard** CPQ em `src/modules/configurador_paineis/` ([RFC](../docs/rfc.pdf)). Demais módulos em `src/modules/` podem ser evolução ERP fora do MVP.

## Comandos

```bash
npm install
npm run dev          # http://localhost:5173
npm test
npm run build
```

Com Docker, use o compose na raiz do repositório (serviço `frontend`).
